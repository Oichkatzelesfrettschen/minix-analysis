================================================================================
CONTACT POINTS FOR ISSUES
================================================================================

If network setup fails:
  - Check TAP support: ip tuntap help
  - Check QEMU version: qemu-system-i386 --version
  - See dev-environment/README.md troubleshooting

If MINIX won't boot:
  - Check ISO: file docker/minix_R3.4.0rc6-d5e4fc0.iso
  - Check logs: ls -lh dev-environment/minix-interactive-*.log
  - See MINIX-Error-Registry.md for boot errors

If serial console unresponsive:
  - Check socat: which socat && socat -V
  - Check socket: ls -l /tmp/minix-serial.sock
  - Reconnect: ./boot-minix-interactive.sh stop && ./boot-minix-interactive.sh start

If measurement tools won't compile:
  - Check MINIX gcc: gcc --version (from MINIX console)
  - Try simple hello world first
  - Check /tmp for source files
