GIT WORKFLOW COMMANDS
================================================================================

Safe commands to use during reorganization:

Move single file:
  git mv old/path/file.md new/path/file.md

Move directory:
  git mv docs/Analysis docs/analysis

Archive multiple files:
  git mv PHASE-*.md archive/phase-reports/
  [will move all matching files]

Check what changed:
  git status

Preview changes:
  git diff --cached

Commit changes:
  git add -A
  git commit -m "docs: consolidate documentation organization"

NEVER use (preserves broken history):
  rm -rf docs/OldName
  mv docs/NewName [wrong - loses git history]
